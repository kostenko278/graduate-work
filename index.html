Как и любую другую программу в ОС Microsoft, «Блокнот» можно отыскать стандартными средствами. Для этого: 1. Кликаем по иконке в виде поиска в левой части панели инструментов Windows. 2. Вводим название программы «Блокнот» или Notepad (для английской версии системы). 3. Нажимаем на клавишу «Ввод» (Enter).

Двойной щелчок мыши – это действие, при котором пользователь два раза быстро нажимает на кнопку мыши. Это один из наиболее распространенных способов взаимодействия с компьютером, который имеет ряд удобных функций и приемуществ. Двойной щелчок мыши используется для запуска программ, открытия файлов, перехода по ссылкам и выполнения других различных действий в операционной системе.

Файлы — это все данные, которые есть в компьютере: документы, фото, музыка, видео и всё остальное. Это объекты, в которых хранится определенная информация. Откройте любой файл на вашем рабочем столе двойным щелчком правой кнопки мыши.
